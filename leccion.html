¡Bienvenido a las Lecciones de C++!

C++ es un lenguaje de programación potente y versátil, ampliamente utilizado en el desarrollo de software, sistemas operativos, juegos, y aplicaciones de alto rendimiento. Aprender C++ te abrirá las puertas a un mundo de posibilidades en la programación.

En estas lecciones, cubriremos los fundamentos de C++, desde los conceptos más básicos hasta la programación orientada a objetos. ¡Prepárate para codificar!

¿Por qué aprender C++?

  • Rendimiento: Permite un control cercano del hardware, ideal para aplicaciones críticas.
  • Versatilidad: Se usa en una amplia gama de dominios.
  • Base sólida: Muchos otros lenguajes se basan en conceptos de C++.
  • Comunidad: Gran cantidad de recursos y una comunidad activa.

1. Conceptos Básicos

Comenzaremos con la estructura básica de un programa en C++, cómo imprimir mensajes en pantalla y el uso de variables.

Tu primer programa: "Hola Mundo"

Todo programa en C++ comienza con la función main(). Aquí te mostramos el clásico "Hola Mundo": 

#include <iostream> // Incluye la librería de entrada/salida

int main() {
    std::cout << "Hola Mundo!" << std::endl; // Imprime "Hola Mundo!" en la consola
    return 0; // Indica que el programa terminó correctamente
}

  • #include <iostream>: Incluye la librería estándar de entrada/salida.
  • int main(): La función principal donde comienza la ejecución del programa.
  • std::cout: Se utiliza para imprimir texto en la consola.
  • <<: Operador de inserción, envía el contenido a cout.
  • std::endl: Inserta un salto de línea y vacía el búfer.
  • return 0;: Indica que el programa finalizó sin errores.

Variables y Tipos de Datos

Las variables son contenedores para almacenar datos. C++ es un lenguaje tipado, lo que significa que debes especificar el tipo de dato que almacenará una variable.

  • int: Números enteros (ej. 10, -5).
  • double: Números de punto flotante (ej. 3.14, -0.5).
  • char: Un solo carácter (ej. 'A', 'b').
  • bool: Valores booleanos (true o false).
  • std::string: Secuencias de caracteres (texto).
#include <iostream>
#include <string> // Necesario para std::string

int main() {
    int edad = 30;
    double precio = 19.99;
    char inicial = 'J';
    bool esActivo = true;
    std::string nombre = "Juan Perez";

    std::cout << "Nombre: " << nombre << std::endl;
    std::cout << "Edad: " << edad << std::endl;
    std::cout << "Precio: " << precio << std::endl;
    std::cout << "Inicial: " << inicial << std::endl;
    std::cout << "Activo: " << esActivo << std::endl; // true se imprime como 1, false como 0

    return 0;
}

2. Estructuras de Control

Las estructuras de control permiten controlar el flujo de ejecución de tu programa, tomando decisiones o repitiendo bloques de código.

Condicionales: if, else if, else

Permiten ejecutar código basado en una condición. 

#include <iostream>

int main() {
    int numero = 10;

    if (numero > 0) {
        std::cout << "El número es positivo." << std::endl;
    } else if (numero < 0) {
        std::cout << "El número es negativo." << std::endl;
    } else {
        std::cout << "El número es cero." << std::endl;
    }

    return 0;
}

Bucles: for, while, do-while

Los bucles se utilizan para repetir un bloque de código varias veces.

Bucle for

#include <iostream>

int main() {
    for (int i = 0; i < 5; ++i) {
        std::cout << "Iteración for: " << i << std::endl;
    }
    return 0;
}

Bucle while

#include <iostream>

int main() {
    int contador = 0;
    while (contador < 3) {
        std::cout << "Iteración while: " << contador << std::endl;
        contador++;
    }
    return 0;
}

3. Funciones

Las funciones son bloques de código reutilizables que realizan una tarea específica. Ayudan a organizar el código y evitar la repetición.

Definición y Llamada a Funciones

#include <iostream>

// Declaración de la función (prototipo)
void saludar(std::string nombre);
int sumar(int a, int b);

int main() {
    saludar("Ana"); // Llamada a la función
    int resultado = sumar(5, 3); // Llamada a la función y almacenamiento del resultado
    std::cout << "La suma es: " << resultado << std::endl;
    return 0;
}

// Definición de la función
void saludar(std::string nombre) {
    std::cout << "Hola, " << nombre << "!" << std::endl;
}

// Definición de la función
int sumar(int a, int b) {
    return a + b;
}

4. Programación Orientada a Objetos (POO)

La POO es un paradigma de programación que se basa en el concepto de "objetos", que pueden contener datos y código. Los pilares de la POO son: encapsulamiento, herencia y polimorfismo.

Clases y Objetos

Una clase es un plano o plantilla para crear objetos, mientras que un objeto es una instancia de una clase. 

#include <iostream>
#include <string>

// Definición de la clase Persona
class Persona {
public: // Miembros públicos (accesibles desde fuera de la clase)
    std::string nombre;
    int edad;

    // Constructor (método especial para inicializar objetos)
    Persona(std::string n, int e) {
        nombre = n;
        edad = e;
    }

    // Método de la clase
    void presentarse() {
        std::cout << "Hola, mi nombre es " << nombre << " y tengo " << edad << " años." << std::endl;
    }
};

int main() {
    // Creación de objetos (instancias de la clase Persona)
    Persona persona1("Carlos", 25);
    Persona persona2("Maria", 30);

    // Llamada a métodos de los objetos
    persona1.presentarse();
    persona2.presentarse();

    return 0;
}

Encapsulamiento

El encapsulamiento consiste en ocultar los detalles internos de un objeto y exponer solo lo necesario a través de una interfaz pública. Se logra usando modificadores de acceso como private y public. 

#include <iostream>
#include <string>

class CuentaBancaria {
private: // Miembros privados (solo accesibles desde dentro de la clase)
    double saldo;

public: // Miembros públicos
    std::string titular;

    CuentaBancaria(std::string t, double s) {
        titular = t;
        saldo = s;
    }

    // Método público para depositar dinero
    void depositar(double cantidad) {
        if (cantidad > 0) {
            saldo += cantidad;
            std::cout << "Depósito de " << cantidad << ". Nuevo saldo: " << saldo << std::endl;
        } else {
            std::cout << "La cantidad a depositar debe ser positiva." << std::endl;
        }
    }

    // Método público para obtener el saldo (getter)
    double getSaldo() {
        return saldo;
    }
};

int main() {
    CuentaBancaria miCuenta("Luis", 1000.0);
    miCuenta.depositar(500.0);
    // miCuenta.saldo = 2000.0; // Esto daría un error, saldo es privado
    std::cout << "Saldo actual: " << miCuenta.getSaldo() << std::endl;
    return 0;
}
